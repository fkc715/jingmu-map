# 京幕地图 Web 端 V1 设计稿

## 文件说明

- 预览文件：`jingmu-web-v1-preview.html`
- 设计稿文件：`jingmu-web-v1-design.md`
- 适用阶段：Web-first 原型与 Taro + React 页面拆分前的 UI 方向确认
- 当前范围：首页总览、北京剧院地图、观剧记录、观剧日历
- 不包含：年度报告显式入口、小程序端页面、票务交易、公开社区

## 页面框架

- 左侧固定侧边栏
- 顶部搜索与全局操作区
- 主内容区单页切换
- 侧边栏页面：
  - 首页总览
  - 北京剧院地图
  - 观剧记录
  - 观剧日历

## 首页总览

### 功能组件

- 全局搜索框
- 筛选按钮
- 添加记录按钮
- 北京剧院地图概览
- 地图状态筛选：
  - 全部
  - 去过

  - 待去

  - 去过剧院

## 页面行为

- 点击侧边栏切换页面
- 每次只显示一个主页面
- 顶部快捷切换按钮与侧边栏保持同步
- 侧边栏 active 状态跟随当前页面
- 不显示年度报告入口
- 观剧记录和图片上传能力合并在记录管理语义下
