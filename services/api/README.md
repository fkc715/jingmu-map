# API Service

This directory is reserved for the Go backend service.

The early backend will use SQLite for local development. The service may later move to PostgreSQL or MySQL if production needs require it.

Planned responsibilities:

- Serve theatre data
- Manage user identity integration
- Store personal theatre visit records
- Track theatre visit counts
- Manage uploaded image metadata
- Support image safety review workflow
- Provide report data for yearly summaries

No backend code has been initialized yet.
